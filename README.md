# scoop-bucket

[discord-assistant](https://github.com/Hyeonjun0527/discord-assistant) Provider Agent 용 Scoop bucket.

```powershell
scoop bucket add nyassistant https://github.com/yeon-intergation-platform/scoop-bucket
scoop install discord-ai-provider-agent
```
`scoop` 이 hash 를 자동 검증하므로 수동 해시검증이 필요 없습니다. 일반 사용자 권한으로 실행됩니다.
릴리스마다 `discord-assistant` 의 CI 가 이 manifest 를 자동 갱신합니다.
